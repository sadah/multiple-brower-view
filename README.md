This is an Electron application that only shows two BrowserViews in one window.

## Setup and Run

Please setup Electron in advance.

https://www.electronjs.org/docs/tutorial/first-app

Install npm modules

```
npm install
```

Run multipul-brower-view

```
npm start
```
